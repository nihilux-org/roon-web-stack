# roon-web-eslint

Shared ESLint configuration for the monorepo.

Configure consuming packages' linting via `extends` in their `.eslintrc.*`:

```json
{
  "extends": "@nihilux/roon-web-eslint"
}
```

For code style rules, see [AGENTS.md](../../AGENTS.md).

For contribution guidelines, see [CONTRIBUTING.md](../../CONTRIBUTING.md).
